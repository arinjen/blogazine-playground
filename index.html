Halo, saya Rizqy Hidayat, 16 tahun. Saya seorang yang menyukai hal-hal sederhana dari hidup sampai desain web. Saya siswa salah satu SMA terkenal di Kota Magelang. Saya tinggal di sebuah desa yang terletak di pojok Kabupaten Magelang, satu-satunya kabupaten yang dikelilingi 5 gunung sekaligus. Jujur, di daerah ini miskin broadband, saya ngenet hanya dari warnet atau hape.

Saya suka menghabiskan waktu di depan komputer atau laptop, ndesain atau ngoding. Saya bukan Web Designer atau Web Developer, tapi saya punya keinginan untuk mendalami bidang tersebut. Saat ini saya belum mempunyai proyek yang dapat dikatakan “sukses”, tapi sudah ada ide untuk mengembangkannya menjadi startup lokal.

Tidur larut sudah menjadi kebiasaan saya, ditemani secangkir kopi dan lagu-lagu dari Jason Mraz. Saya suka rambut panjang. Sayang peraturan sekolah berbanding terbalik dengan kesukaan saya. Sudah beberapa kali kena potong sama guru. Tapi tetap nggak kapok :D

Saya juga mempunyai hobi bertualang, hiking, dan climbing. Walau begitu, saya masih pemula di bidang ini. Belum banyak gunung yang saya daki, baru sekitar 4 gunung. Saya tergabung dalam Glacial (Gladiool Pecinta Alam), salah satu ekstrakulikuler di sekolah saya. Dari namanya sudah bisa ditebak dimana saya sekolah, hehe.

Untuk mengenal saya lebih jauh, silahkan bertandang ke bagian kontak.

Awalnya saya membuat blog ini hanya untuk mencicipi blogging dengan rasa web statis dengan bahan dasar Jekyll dan Github. Kemudian saya tertular virus blogazine dan merasakan sangat mudah menerapkan konsep blogazine dengan mesin ini. Awalnya hanya beberapa blogazine saja, tapi setelah saya beli domain, saya pindah haluan, membuat blog ini full blogazine.

Ide nama “Playground” datang begitu saja dan akhirnya saya membuatnya menjadi sub-domain dari situs saya. Domain baru, desain baru. Tetap mengusung tema simpel dan minimalis, dengan warna hijau dan biru.

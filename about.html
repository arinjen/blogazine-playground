tentang saya

Nama saya Rizqy Hidayat. Saya siswa sebuah SMA negeri di Magelang. Tinggal di kaki Gunung Telomoyo dan Gunung Andhong. Saya suka web design dan web developing dengan aliran simple dan minimalis.

Pertama melihat HTML/CSS kelas 7, SMP. Mulai semakin dalam saat kelas 9. Sampai saat ini saya sudah pernah menggunakan beberapa mesin, antara lain Textpattern, Wordpress, Blogger, Tumblr, Codeigniter, Panada, GetSimple, WolfCMS, Joomla, dan lain-lain. Tapi saya lebih cinta webstatis dengan Jekyll (yang digunakan untuk blog ini).

Selain web design, saya juga suka hiking, adventuring, dan lain-lain. Tidak dapat dipungkiri, saya juga seorang "mrazer", sebutan untuk penggemar Jason Mraz. Beberapa lagu yang saya sukai dari Mr. A-Z: I'm Yours, The Remedy (I won't worry), Clockwatching, Sleep All Day, Make it Mine, You and I Both, No Stopping Us, Wordplay, dan masih banyak lagi :D

tentang blog ini

Blog ini belum lama berdiri. Saya men-dokumentasikan eksperimen-eksperimen yang pernah saya coba. Ini adalah webstatis. Yeah, simple, mudah, gratis!. Bermesin Jekyll dan Github sebagai host-nya. Belum sempat beli domain karena keterbatasan dana :D

Seperti saya katakan di atas, design blog ini sangat simple. Background hanya garis vertikal berulang, putih. Warna-warni. Serta menggunakan beberapa trik CSS, seperti segitiga, border dalam, dan media query.

Satu lagi, blog ini sudah responsive.

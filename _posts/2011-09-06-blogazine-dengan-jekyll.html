--- layout: post title: Blogazine di Webstatis dengan Jekyll ---

Beberapa hari yang lalu, saya melihat status facebook yang berisi bagaimana ber-blogazine ria di webstatis dengan Jekyll. Kemudian saya terpikir untuk membuat plugin untuk mempermudah dalam membuat style blogazine. Pemikiran saya, bagaimana caranya untuk:

  • Memasukkan nama file css dalam postingan dengan memanfaatkan YAML front block
  • Memasukkan element <link> dalam setiap postingan yang memiliki style blogazine

    • Berangkat dari pemikiran tersebut, saya mulai mempelajari struktur plugin Jekyll. Bingung! Lalu saya mencari referensi dan menemukan tulisan tentang bagaimana membuat meta tag desccriptin untuk setiap post. Ternyata code-nya sangat simple.

    Penerapan

    1. Menambahkan Liquid Tag pada layout untuk post

      Saya menggunakannya seperti ini dan menempatkannya di bawah stylesheet utama:

      {% highlight ruby %} {{ "{% if page.style "}}%} {{ "{% endif "}}%} {% endhighlight %}

      Variabel page.style adalah salah satu dari YAML front block pada setiap awal post. {{ "{{ page.style "}}}} menyatakan "jika dalam YAML front block variabel style didefinisikan, maka element <link> akan dibuat"

    2. Copy file css

      Copy file css ke dalam folder css. Pastikan nama file sama dengan yang ditulis pada YAML front block untuk style.

    3. Menulis post

      Berikut adalah contoh YAML front block dengan style blogazine-test:

      {% highlight ruby %} --- layout: post title: Testing Blogazine Style di Jekyll style: blogazine-test --- {% endhighlight %}

      Pastikan value style sama dengan nama file css yang ada.

    Yup, that's all. Hanya itu. Simple dan mudah :D
